## 二、推荐技能组合

### 组合1: 全栈研究 → 开发 → 部署

```
deer-deep-research (研究)
        ↓
deer-data-analysis (分析数据)
        ↓
deer-frontend-design / deer-web-design (设计)
        ↓
deer-vercel-deploy (部署)
```

**触发词**: "做一个XX网站"
**技能链**: 研究 → 数据分析 → 前端设计 → 部署

---


## 五、TDD 开发模式

结合 tdd-workflow 使用:

```
1. deer-consulting (需求分析)
        ↓
2. tdd-workflow (TDD流程)
   - RED: 先写测试
   - GREEN: 最小实现
   - REFACTOR: 重构
        ↓
3. react-best-practices (最佳实践)
        ↓
4. deer-vercel-deploy (部署)
```

   - last30days + deer-deep-research

6. **检测到设计/UI**: 
   - deer-frontend-design + deer-web-design + deer-chart-visual

7. **检测到内容创作**:
