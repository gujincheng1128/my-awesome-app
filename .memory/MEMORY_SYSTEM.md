## 记忆召回机制

### 召回触发词

| 触发词 | 记忆层级 | 说明 |
|--------|----------|------|
| "我之前说过..." | L2 | 情景记忆召回 |
| "记得上次..." | L2 | 会话历史召回 |
| "我的偏好是..." | L3 | 用户偏好读取 |
| "上次做的项目..." | L2 | 项目记忆召回 |
| "如何实现XX..." | L4 | 程序记忆召回 |
| "XX是什么..." | L3 | 概念定义召回 |

### 召回流程

```
用户输入
    ↓
┌─────────────────────────────────────────┐
│ 意图识别 → 触发记忆召回?                 │
└─────────────────────────────────────────┘
    ↓ 是
┌─────────────────────────────────────────┐
│ 选择召回层级                             │
│ • L1: 工作记忆 (当前会话)               │
│ • L2: 情景记忆 (会话历史)               │
│ • L3: 语义记忆 (概念事实)               │
│ • L4: 程序记忆 (技能模式)               │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│ 检索匹配 → 排序 → 返回                   │
└─────────────────────────────────────────┘
    ↓
整合到响应上下文
```

## 记忆优化

### 自动优化规则

```markdown

1. **会话结束**
   - 自动保存情景记忆
   - 提取关键信息
   - 更新用户偏好

2. **每日整理**
   - 合并相似记忆
   - 清理过期内容
   - 更新索引

3. **每周回顾**
   - 总结学习成果

4. **每月归档**
   - 归档旧会话

   - 生成统计报告
```
