# Autoresearch - 自主研究 Agent

## 概述

Autoresearch 是一个自主研究 Agent，能够自动搜集信息、分析数据、综合结论，帮助用户完成复杂的研究任务。

## 核心功能

- **自动搜集**: 从多个来源搜集相关信息
- **智能分析**: 分析信息的相关性和准确性
- **综合报告**: 生成结构化的研究报告
- **引用追踪**: 记录信息来源和引用
- **迭代研究**: 根据发现持续深入研究

## 工作流程

1. **理解任务**: 分析研究目标和要求
2. **制定计划**: 制定信息搜集计划
3. **搜集信息**: 从网络、文档等来源搜集
4. **分析整理**: 整理和分析搜集的信息
5. **生成报告**: 综合生成研究报告

## 使用场景

- 市场调研
- 竞品分析
- 学术研究
- 技术调研
- 新闻事件追踪
