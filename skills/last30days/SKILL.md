# last30days - AI 工具趋势追踪

## 概述

last30days 是一个追踪最近 30 天热门 AI Agent 和工具的技能，帮助用户了解 AI 领域的最新动态和趋势。

## 核心功能

- **热门追踪**: 追踪 GitHub Trending 的 AI 项目
- **新工具发现**: 发现新发布的有趣工具
- **趋势分析**: 分析工具使用趋势
- **分类整理**: 按类别整理热门项目

## 追踪类别

1. **AI Agent 框架**: 新的 Agent 框架和库
2. **开发工具**: 提升开发效率的 AI 工具
3. **研究项目**: 有影响力的研究项目
4. **开源模型**: 新发布的大模型
5. **应用案例**: 创新的 AI 应用

## 使用方式

```bash
# 列出最近 30 天的热门 AI Agent
/openclaw skill use last30days --list agents

# 搜索特定类别的工具
/openclaw skill use last30days --search "code generation"

# 获取趋势报告
/openclaw skill use last30days --report weekly
```

## 数据来源

- GitHub Trending

- Reddit r/MachineLearning
