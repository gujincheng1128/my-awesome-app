# Metis - 预规划顾问 Agent

## 概述

Metis 是 Oh My OpenCode 的预规划顾问 Agent，在任务执行前进行分析，识别模糊性、隐藏意图和潜在的 AI 优化机会。

## 核心职责

- **需求澄清**: 识别任务描述中的模糊点
- **意图挖掘**: 发现用户的真实需求
- **风险识别**: 识别潜在的执行风险
- **方案建议**: 提供最佳执行方案建议
- **AI 优化**: 识别可由 AI 改进的环节

## 分析维度

1. **任务清晰度**: 任务是否明确定义
2. **技术可行性**: 现有技术能否实现
3. **资源需求**: 需要什么资源和工具
4. **风险评估**: 执行过程中的潜在风险
5. **优化机会**: 可自动化或改进的地方

## 与其他 Agent 配合

- **Sisyphus**: 接收 Metis 的分析结果进行规划
- **Hephaestus**: 根据 Metis 的建议执行开发
- **Argus**: 参考 Metis 的风险评估进行审查
