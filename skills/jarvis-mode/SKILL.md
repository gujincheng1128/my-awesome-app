## 记忆层级

1. **L1 工作记忆** - 当前会话上下文
2. **L2 情景记忆** - 重要会话摘要
3. **L3 语义记忆** - 知识概念事实
4. **L4 程序记忆** - 技能工作流模式

## TDD 循环

```
🔴 RED → 🟢 GREEN → 🔵 REFACTOR → (循环)
```

## Agentic Patterns

- **规划**: 任务分解、子目标生成、计划验证
- **执行**: 工具调用、渐进执行、错误恢复
- **反思**: 结果验证、自我批评、知识整合
- **协作**: 多Agent协调、任务委托、结果聚合
