# Atlas - 知识导航 Agent

## 概述

Atlas 是一个知识导航 Agent，帮助用户在大型知识库中浏览、搜索和组织信息。

## 核心功能

- **智能导航**: 在复杂知识结构中导航
- **快速搜索**: 高效检索相关知识
- **知识整理**: 组织和分类知识条目
- **关联发现**: 发现知识之间的关联
- **学习建议**: 推荐学习路径和资源

## 使用场景

- 大型代码库理解
- 项目文档浏览
- 知识库探索
- 学习路径规划
- 信息发现

## 导航方式

1. **主题浏览**: 按主题分类浏览
2. **关键词搜索**: 精确关键词匹配
3. **关联跳转**: 跟随知识关联导航
4. **推荐发现**: 根据当前内容推荐相关知识
