# OpenCode Skill: opencode-superpowers

- Summary: A scaffolded plugin package for OpenCode with an initial smart-commit submodule.
- Usage: Enable via OpenCode UI or run the script: `./opencode-superpowers.sh smart-commit` when testing.

## Submodules
- smart-commit: Generates atomic commit messages from staged changes.
- Usage: `opencode-superpowers smart-commit.sh [-c|--commit]` to auto-commit after generating message.
